## dbt-athena 1.11.0-b1 - May 26, 2026

### Features

- Add STS AssumeRole support for cross-account access ([#1657](https://github.com/dbt-labs/dbt-adapters/issues/1657))
- Added support for --empty in dbt seed ([#1865](https://github.com/dbt-labs/dbt-adapters/issues/1865))

### Fixes

- Fix infinite loop when Athena Spark session terminates unexpectedly during calculation submission ([#1507](https://github.com/dbt-labs/dbt-adapters/issues/1507))
- Fix table and column description truncation to respect Athena limits by using ellipsis_comment helper ([#1460](https://github.com/dbt-labs/dbt-adapters/issues/1460))
- Fix bucket partitioning to group partition conditions with OR before combining with bucket conditions via AND, reducing redundant query expansion. ([#1636](https://github.com/dbt-labs/dbt-adapters/issues/1636))
- Override check_schema_exists in Athena adapter to use Glue API ([#1650](https://github.com/dbt-labs/dbt-adapters/issues/1650))
- equals macro now uses IS NOT DISTINCT FROM clause for comparison ([#1663](https://github.com/dbt-labs/dbt-adapters/issues/1663))
- When persist_docs.columns is enabled, dbt now warns about columns defined in YAML that don't exist in the database via the validate_doc_columns helper. ([#1690](https://github.com/dbt-labs/dbt-adapters/issues/1690))
- Fix get_partition_batches exceeding partition limit when combining non-bucket and bucket partitions ([#1783](https://github.com/dbt-labs/dbt-adapters/issues/1783))
- Fixed a jinja syntax issue in athena__persist_docs macro ([#NA](https://github.com/dbt-labs/dbt-adapters/issues/NA))

### Under the Hood

- Optimize Athena partition deletions for insert_overwrite strategy ([#1125](https://github.com/dbt-labs/dbt-adapters/issues/1125))

### Dependencies

- Bump dbt-adapters floor to 1.24.1 for catalogs v2 and JS UDF support ([#1967](https://github.com/dbt-labs/dbt-adapters/pull/1967))
- Allow pre-release versions of dbt-core to be selected so release integration tests pick up new dbt-core features ([#1976](https://github.com/dbt-labs/dbt-adapters/pull/1976))
