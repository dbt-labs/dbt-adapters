## dbt-adapters 1.22.10 - March 30, 2026

### Features

- Add packages support to default__get_function_python_options() so adapters can template PyPI packages into Python UDF CREATE statements ([#1737](https://github.com/dbt-labs/dbt-adapters/issues/1737))

### Fixes

- When persist_docs.columns is enabled, dbt now warns about columns defined in YAML that don't exist in the database and filters them out before calling alter_column_comment. The new validate_doc_columns helper eliminates a duplicate get_columns_in_relation roundtrip per model. ([#1690](https://github.com/dbt-labs/dbt-adapters/issues/1690))
- Fix unit test fixture string values being silently truncated when database column has a narrow varchar type ([#1806](https://github.com/dbt-labs/dbt-adapters/issues/1806))
