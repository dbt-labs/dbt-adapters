## dbt-bigquery 1.11.0 - December 22, 2025

### Features

- Add support for BigQuery resource tags via `resource_tags` configuration option ([#565](https://github.com/dbt-labs/dbt-adapters/issues/565), [#577](https://github.com/dbt-labs/dbt-adapters/issues/577))
- Support the updating of incremental model description after initial creation ([#585](https://github.com/dbt-labs/dbt-adapters/issues/585))
- Add `enable_change_history` option for models on BigQuery. ([#1206](https://github.com/dbt-labs/dbt-adapters/issues/1206))
- Add support for scalar functions ([#1290](https://github.com/dbt-labs/dbt-adapters/issues/1290))
- Support BigQuery STRUCT schema change ([#599](https://github.com/dbt-labs/dbt-adapters/issues/599))
- Add support for python UDFs ([#1410](https://github.com/dbt-labs/dbt-adapters/issues/1410))
- Add support for aggregate sql functions ([#1432](https://github.com/dbt-labs/dbt-adapters/issues/1432))
- :bug: :snowman: Support batched source freshness in bigquery using TABLE_STORAGE, behind behavior change flag. ([#561](https://github.com/dbt-labs/dbt-adapters/issues/561))
- Add support for py3.13 ([#1475](https://github.com/dbt-labs/dbt-adapters/issues/1475))
